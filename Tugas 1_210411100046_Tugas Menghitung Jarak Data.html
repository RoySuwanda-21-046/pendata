1. Tipe Data

1.1 Atribut Nominal / Kategorical

Nominal berarti “berkaitan dengan nama.” Nilai atribut nominal adalah simbol atau nama benda. Setiap nilai mewakili beberapa jenis kategori, kode, atau status, sehingga atribut nominal juga disebut sebagai kategori. Nilai tidak memiliki urutan yang berarti. Dalam ilmu komputer, nilai juga dikenal sebagai pencacahan.

1.2 Atribut Binary (Biner)

Atribut biner adalah atribut nominal dengan hanya dua kategori atau status 0 atau 1, di mana 0 biasanya berarti bahwa atribut tersebut tidak ada, dan 1 berarti ada. Atribut biner disebut sebagai Boolean jika kedua status sesuai dengan benar dan salah.

1.3 Ordinal

Atribut ordinal adalah atribut dengan kemungkinan nilai yang memiliki urutan atau peringkat yang bermakna di antara mereka, tetapi besarnya antara nilai-nilai yang berurutan tidak diketahui.

1.4 Numeric

Atribut numerik bersifat kuantitatif; yaitu, itu adalah kuantitas yang dapat diukur, direpresentasikan dalam bilangan bulat atau nilai nyata. Atribut numerik dapat berskala interval atau berskala rasio.

  • Menghitung Jarak menggunakan metode Euclidean Distance

Euclidean Distance dapat digunakan untuk menghitung jarak euclidean antara dua data dengan menghitung akar kuadrat dari selisih kuadrat antara nilai atribut Gaji.

Rumus Euclidean Distance: jarak(i,j) = sqrt((Gaji_i - Gaji_j)^2)

Keterangan: * jarak(i,j) : Jarak dissimilarity antara Gaji responden i dan j

* Gaji_i : Nilai Gaji responden i

* Gaji_j : Nilai Gaji responden j

Rumus tersebut dapat dijelaskan sebagai berikut:

* Ambil nilai Gaji responden i dan j

* Hitung selisih antara nilai Gaji responden i dan j

* Kuadratkan selisih tersebut

* Hitung akar kuadrat dari hasil kuadrat pada langkah 3

* Hasil pada langkah 4 merupakan nilai jarak dissimilarity antara Gaji responden i dan j

# Baca data dari Tabel
data = df.head()

# Ambil data Gaji dari dua data yang ingin dihitung jarak dissimilaritynya
Gaji1 = data.loc[data['nama'] == 'roy']['gaji'].iloc[0]
Gaji2 = data.loc[data['nama'] == 'suwanda']['gaji'].iloc[0]

# Hitung jarak dissimilarity gaji menggunakan Euclidean Distance
jarak = math.sqrt((Gaji1 - Gaji2)**2)

# Tampilkan hasil perhitungan
print('Jarak dissimilarity Gaji antara roy dan suwanda adalah:', jarak)
